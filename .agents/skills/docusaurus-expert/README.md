# docusaurus-expert

A skill for building fast, SEO-optimized static sites with Docusaurus v3.9.2 using Markdown/MDX, SEO metadata, and plugins. Helps with setup, docs, SEO optimization, plugin integration, and GitHub Pages deployment.

## Usage

Load this skill when working on:
- Docusaurus configuration, plugins, or theme customization
- SEO metadata, OpenGraph tags, or social card setup
- GitHub Pages deployment and CI/CD workflows
- Build errors, broken links, or performance issues
- Markdown/MDX content authoring with frontmatter
